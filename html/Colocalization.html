Colocalization

The function Colocalization calculates colocalization of two genes/cell types by assessing the closest neighbors to each cell/gene and comparing with a set of 100 simulation. The real results are compared with the simulation leading to a Z-score that indicates colocalization (over 0) or mutual exclusion (below 0)
Input: Matisse object (it can be both cells or bins)
Parameters: Matisse-Colocalization_number_neighbors: number of neighbors to find from each cell/spot.
Optional input: in case more than one samples is analyzed, specify the sample number ie. '3'. Default is sample 1.
Output: Colocalization table
Example
Here you have the example of how to use the function. We could calculate it with spots, cells or bins. In this case we use Bins. First we load the bins, summarized in a matisse object:
load ('data\Matisse objects\bins_mousebins_clustered.mat');
Some parameters can be modified:
BINS.Colocalization_number_neighbors=3;
Then we run the function. After the analyis appears, MATLAB will ask to specificy the number of PC that one wants to keep ( like [1,2,6,7,8] or [1:end])
Colocalization_table=Colocalization(BINS);
Colocalization calculation is starting. Note that it can take several hours...
