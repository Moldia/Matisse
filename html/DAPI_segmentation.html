DAPI_segmentation

The function DAPI_segmentation segments a DAPI image and obtains cell boundries after a watershed segmentation
Input: Matisse object
Parameters in Matisse:
-Matisse.segmentation percent: Assigns all values of an image above a percentile of the image intesity to a true DAPI signal. It'is 98 by default.
Output: Matisse object with Cellmap and centroid coordinates.
Example
Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\Spots_mousebrain.mat');
There are few parameters that can be customized,like segmentation percent, which indicates the threshold intensity (in percentage) for considering a pixel as a possible cell:
SPOTS.segmentation_percent=98;
Then we run the function
SPOTS=DAPI_segmentation(SPOTS);
