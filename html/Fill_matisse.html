Fill_matisse

The function fill_matisse integrates the spatial information of different samples into a single object, aiming to standardize a format common in the rest of the analysis, the Matisse object.
Input: A list object containing different objects including the information of each sample (one for each sample)
Output: A Matisse object, the standard object used for ISS analysis in this toolbox.
Previous steps
Here you have the previous steps you need to do before using the function.
%We load all the information of the first sample on on SPOTS1
decoded_file = 'data\humanbrain\ISS_h_brain_03_PreliminaryCalls.csv';
image = 'data\humanbrain\HCA_09_hm_120_c1_stitched-1.tif'; % important for size
scale=1; %
SPOTS1=ISS_getspots(decoded_file,'genecolumn',1,'XYcolumn',[2,3]);
SPOTS1.image=image;
SPOTS1.scale=scale;
%We load all the information of the second sample on SPOTS2
decoded_file = 'data\humanbrain\HCA10_1_genes_and_coordinates.csv';
image = 'data\humanbrain\HCA_10_1_hm_133_c1_stitched-1.tif.mat.tif'; % important for size
scale=1; %
SPOTS2=ISS_getspots(decoded_file,'genecolumn',1,'XYcolumn',[2,3]);
SPOTS2.image=image;
SPOTS2.scale=scale;
%And then we combine them into a big object (note that if you have added more samples you need to add them here as well)
ALLSPOTS.SPOTS1=SPOTS1;
ALLSPOTS.SPOTS2=SPOTS2;
This is the example of how to use fill_matisse
SPOTS=fill_matisse(ALLSPOTS);
