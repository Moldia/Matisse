GeneDensity

The function GeneDensity plots the KDE gene density of genes specified
Input: Matisse object
Parameters: Some parameters can be modifiable within the Matisse object:
- SPOTS.GeneDensity_Genes_of_interest: genes to be ploted
- SPOTS.GeneDensity_bandwidth:bandwidth for the KDE;
- SPOTS.GeneDensity_colormap: colormap used in the plot
Output: Plot of the two KDE in red and green
Example
Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\Spots_mousebrain.mat');
We need to specify the genes to be visualized. Some other parameters can be modified as well.
SPOTS.GeneDensity_Genes_of_interest={'Slc17a7','Gad1'};
SPOTS.GeneDensity_bandwidth=200;
SPOTS.GeneDensity_colormap=hot;
Finally we run the function:
GeneDensity(SPOTS);
