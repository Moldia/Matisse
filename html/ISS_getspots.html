ISS_getspots

The function ISS_getspots reads a typical ISS output, containing the coordinates in XY for all the genes detected in a sample, and saves it in a list object (SPOTS1). This function expects the genename to be in the second column of a .csv and XY to be on the 3rd and 4th column respectively.
Input: Pathway to a csv
Parameters: the column position of gene names, as well as the columns position of XY coordinates in the input matrix needs to be specified
Output: List object containing all the gene names and coordinates.
Here we can find an example of how to use this function:
decoded_file = 'data\mousebrain\QT_0.2_details_noNNNN.csv';
image = '\data\mousebrain\wholeimage.tif';
SPOTS1=ISS_getspots(decoded_file,'genecolumn',2,'XYcolumn',[3,4]);
