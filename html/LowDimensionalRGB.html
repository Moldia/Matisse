LowDimensionalRGB

The function LowDimensionalRGB is used to represent the complexity of cell expression in an RGB schema, by representing just the top 3 components of the tSNE done on the expression of cells/bins
Input: - Matisse object (it can be both cells or bins)
- Method used ('umap','tsne' or 'pca')
- option for plotting or not the expression of every gene ('genexpression','no')
Parameters in Matisse:
-Matisse.count_threshold: minimum expression that a cell/bin needs to be included on the analysis.
Output: Figure representing the top 3 tsne compnents in RGB
Example
Here you have the example of how to use the function. First we load the bins, summarized in a matisse object
load ('data\Matisse objects\bins_mousebins.mat');
There are few parameters that can be customized:
BINS.count_threshold=5;
Then we run the function. Three methods can be used for LowDimensional representation in RGB, including 'umap','tsne' and 'pca'. This method needs to be specified in the second position. In the third position we need to specifiy wether one wants the low dimensional plot for the expression of individual genes ( by specifying 'genexpression') or not (by adding on the third term a 'no')
. NOTE THAT: You need to select genes that you want to include in the analysis.
LowDimensionalRGB(BINS,'umap','no');
