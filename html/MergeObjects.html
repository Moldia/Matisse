MergeObjects

The function MergeObjects merges two Matisse objects.
Input: two Matisse objects
Output: a combined MatisseMOD object

Example

Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\Spots_mousebrain.mat');
load ('data\Matisse objects\Spots_mousebrain2.mat');
We set the Genes_of_interest and the bandwidth.Then we just call the function giving as an input a matisse object.
CORTEX_SPOTS=MergeObjects(SPOTS_2ND,SPOTS);
Unrecognized function or variable 'SPOTS_2ND'.
