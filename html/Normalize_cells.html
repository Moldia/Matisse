Normalize_cells

The function normalize_cells filters the cells/bins using different methods
Input: Matisse object (it can be both cells or bins)
Matisse parameters: some parameters need to be adjusted before filtering:
-NC_method: normalization method, can be 'standarize','log2' or 'log10'
Output: Matisse object with normalized data
Example
Here you have the example of how to use the function. First we load the bins, summarized in a matisse object
load ('data\Matisse objects\bins_mousebins.mat');
Then we adjust the parameters starting with NC:
BINS.NC_method='standardize';
Then we run the function
BINS_Normalized=Normalize_cells(BINS);
