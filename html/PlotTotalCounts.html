PlotTotalCounts

The function PlotTotalCounts represents the frequency of each gene in raw ISS data. It can be used to compare the expression of two different samples, in case they are both included in the same MatisseMod object.
Input: Matisse object
Output: Plots showing the frequency of each gene
Example
Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\Spots_mousebrain.mat');
Then we apply the function:
PlotTotalCounts(SPOTS);
