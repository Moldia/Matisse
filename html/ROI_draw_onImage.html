ROI_draw_onlmage

The function ROI_draw_onImage selects a subset of the sample analysed by drawing a desired polygon on the area to be selected.
Input: Matisse object + sample selected from the ones included in Matisse (ej.1)
Optional input: Gene selected to display density and select a ROI based on that.
Output: A Matisse object including only the subset of the genes

Example

Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\Spots_mousebrain.mat');
We can select a Region of interest just inputing the Matisse object. We select a part of the 1st of the samples included in the Matisse object.
Note that: when running the script will ask to draw a polygon. When finnish drawing the polygon, just double click on the image and close it. It will also ask for an append for the new image saved. The append needs to be added between quotation marks (eg. 'cortex')
[ROI_cortex]=ROI_draw_onImage(SPOTS,1)
It's also possible to add as a 3rd argument a gene and then draw the polygon based on density.
GENE={'Syt1'};
[DCNPOSITIVE_REGION]=ROI_draw_onImage(SPOTS,1,GENE)
