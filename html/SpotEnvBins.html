SpotEnvBins

The function SpotEnvBins is designed for binning the expression map found in a matisse object by finding the expression enviroment of each spot in the dataset..It's specially useful if segmentation is not possible.
Input: Matisse object
Parameters in Matisse:
- Matisse.SpotEnvBins_radius: radius of each bin, in pixels.
Output: Matisse object representing the bin coordinates and their expression.
Example
Here you have the example of how to use the function. First we load the spots, summarized in a matisse object
load ('data\Matisse objects\Spots_mousebrain.mat');
There are few parameters that can be customized:
SPOTS.SpotEnvBin_radius=50;
Then we run the function. This process can be time consuming (1-10h) and generates a big dataset to work with, yet meaningful.
ENV_BINS=SpotEnvBin(SPOTS);
