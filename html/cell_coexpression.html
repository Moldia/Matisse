cell_coexpression

The function cell_coexpression uses binned data (bins/cells) to calculate the presence of a specific gene in the cells that present another specific gene. This is done for the combination of all the genes in the dataset,
Input: Matisse object (it can be both cells or bins)
- image selected ie. '1'
Output: Coexpression figures
Example
Here you have the example of how to use the function. We could calculate it with spots, cells or bins. In this case we use Bins. First we load the bins, summarized in a matisse object:
load ('data\Matisse objects\bins_mousebins_clustered.mat');
Then we run the function
cell_coexpression(BINS,1)
