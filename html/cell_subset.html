Cell_subset

The function cell_subset selects a subset of cells/clusters for further analysis within your matisse object
Input: Matisse object
Optional input: cells/clusters selected
Output: A Matisse object including only the subset of the cells/clusters
Example
Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\bins_mousebins_clustered.mat');
We can select the genes without any previous list just inputing the Matisse object
SUBSET_BINS=cell_subset(BINS);
