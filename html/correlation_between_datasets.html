Correlation between datasets

The function correlation_between_datasets compares the clusters/cell types defined in two different cell types. This can bet two ISS samples or one ISS sample and a SCdata
Input: -Matisse object with clusters defined
-Matisse object witth clusters defined
Output: Object showing the correlation between datasets
Example
Here you have the example of how to use the function. First we load the data:
load ('data\Matisse objects\bins_mousebins_clustered.mat');
Then we run create a MATISSE object with the single cell RNA seq reference data, in this case including Single cell data from Allen brain Atlas:
path='data\Matisse objects\SCDATA_Reference.csv';
[SINGLE_CELL]=load_scRNAseq(path);
Finally we run compare the expression in both objects:
correlation_between_datasets(BINS,SINGLE_CELL);
