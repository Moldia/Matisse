Define_cells

The function define_cells combines the position of cells in the DAPI image and the gene position to define a matisse object defining the expression of each cell
Input: Matisse object contating the spots coordinates and Cellmap
Output: Matisse object defining Cells
Example
Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\Spots_mousebrain.mat');
Then we run the function
CELLS=define_cells(SPOTS);
