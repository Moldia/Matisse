Expression_clusters

The function expression_clusters allows the visualization of the mean expression of the different clusters defined previously
Input: Matisse object (it can be both cells or bins) previously clustered
- method used to visualize ('violin' or 'boxplot')
Output: Matisse object with clustered data
Example
Here you have the example of how to use the function. First we load the bins, summarized in a matisse object, already clustered
load ('data\Matisse objects\bins_mousebins_clustered.mat');
Then we run the function. NOTE THAT: We need to select the clusters we want to visualize.
expression_clusters(BINS,'violin');
