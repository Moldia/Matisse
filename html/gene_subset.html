Gene_subset

The function gene_subset selects a subset of genes for further analysis within your matisse object
Input: Matisse object
Optional input: List of genes to be selected in a cell array
Output: A Matisse object including only the subset of the genes
Example
Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\Spots_mousebrain.mat');
We can select the genes without any previous list just inputing the Matisse object
[SUBSET_SPOTS]=gene_subset(SPOTS)
