Gradients

The function gradients represents the density of cells/genes in a sample in 1D from selected spots
Input: Matisse object + sample selected from the ones included in Matisse (ej.1)
Parameters in Matisse:
-Matisse.Gradients_bandwidth: bandwidth of the KDE representation can be customized
- Matisse.Gradients_limits: represents the maximum distance from the origin to consider in the plot.
Output: Gradient plots
Example
Here you have the example of how to use the function. First we load the data
load ('data\Matisse objects\Spots_mousebrain.mat');
There are few parameters that can be customized:
SPOTS.Gradients_bandwidth=200;
SPOTS.Gradients_colormap=parula;
SPOTS.Gradients_limit=10000;
Then we run the function. NOTE THAT: To define the origin of the gradient, a polygon needs to be drawn in the image
gradients(SPOTS,1);
