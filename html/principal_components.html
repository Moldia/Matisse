Principal_components

The function principal_components explores the principal components in our dataset and allows for filtering
Input: Matisse object (it can be both cells or bins)
Output: Matisse object with reduced dataset
Example
Here you have the example of how to use the function. First we load the bins, summarized in a matisse object
load ('data\Matisse objects\bins_mousebins.mat');
Then we run the function. After the analyis appears, MATLAB will ask to specificy the number of PC that one wants to keep ( like [1,2,6,7,8] or [1:end])
BINS_Normalized=principal_components(BINS);
